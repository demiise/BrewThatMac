# Changelog

## 0.1.0 - 2026-02-20

- Added `brewthatmac.sh` dispatcher (`up`, `dump`, `drift`, `help`).
- Added `brewup.sh` for brew/mas update, cleanup, and doctor logging.
- Added `brewdump.sh` for Brewfile dump + version snapshots.
- Added `brewdrift.sh` for drift detection and guided reconciliation.
- Added `.env`-based configuration support with `.env.example`.
- Added publish-ready metadata: `README.md`, `.gitignore`, `LICENSE`.
